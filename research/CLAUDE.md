# research/ — Library

Source material + extracted knowledge. Grows with Oren.

## Structure
- `books/` — source PDFs. **Never modify.** Read-only.
- `insights/` — extracted markdown + metadata (from `/markpdf` or manual notes)
- `external/` — pasted articles, research dumps, external references

## Rules
- Every `insights/*.md` file cites its source in frontmatter (title, author, extracted date)
- `/markpdf` is the default extraction tool
- `external/` files prefix with date: `2026-04-23-source-slug.md`
- When research informs content, link the insight file from the draft
